# Agent Skill: Professional Portfolio Generator (HTML/CSS)

## Architectural Constraints
- **Platform Limitation:** Strictly static HTML5 and vanilla CSS3 only. No PHP, no external JS frameworks. GitHub Pages compatibility is mandatory.
- **Navigation:** Single-page application (SPA) layout or minimalist tabs. Maximum flat hierarchy; absolutely no nested sub-menus or multi-level dropdowns.
- **Performance & Safety:** Zero heavy animations, video backgrounds, or complex visual scripts to guarantee instant loading times.

## Design System (Anti-AI-Cliché)
- **Vibe:** Minimalist, high-contrast dark mode inspired by Carrd structures.
- **Colors:** Deep background (`#121212`), high-readability body text (`#e0e0e0`), professional accent color (e.g., steel blue or muted slate). No primary-color button loops.
- **Typography:** Modern, clean system sans-serif stack (`Inter`, `system-ui`, `-apple-system`). Never use standard generic fonts like Arial or Times New Roman.

## Mandatory Content Layout & Ordering

1. **Hero Section (Startseite)**
   - Minimalist header with full name, target professional role, and a high-quality professional portrait placeholder.
   - A crisp, 2-sentence value proposition detailing the core objective.

2. **Core Profile (Über mich / Kurzprofil)**
   - A highly tailored, impact-focused professional summary. 
   - Balance point: Specific enough to showcase massive experience, adaptable enough for multiple enterprise targets.

3. **Professional Milestones (Lebenslauf)**
   - Clean, chronological or functional timeline of career stations.
   - Dedicated sections for hard skills, framework/infrastructure expertise, and language fluencies.

4. **Resource Link (Downloadbereich)**
   - A primary, high-visibility call-to-action (CTA) button linking to the unified digital application bundle: `https://1drv.ms/f/c/5277166773def518/IgAKEDSyV4KVQ5mLOdQ_6CK8Afhcjr294D3zv1TYQAk4vag`
   - Explicit labeling: "Komplette Bewerbungsmappe als PDF herunterladen".

5. **Contact & Footer**
   - Direct display of active contact data (Full Name, Address, Email, Phone). No hidden contact forms that require extra clicks.
